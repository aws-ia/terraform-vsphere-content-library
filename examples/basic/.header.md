# Basic example

If deployed with the default values, this example will create a [content library][content_library] named `example-content-library` backed by a [datastore] named `WorkloadDatastore`, and one ISO [item][items] named `vmware-tools-windows-11.3.0-18` in your [VMware Cloud on AWS][vmconaws] or [VMware vSphere][vsphere] on-premises environment.
